## ğŸ“– Introduction

**AutismRecover** is a Human-Centric Computing project designed to bridge the social interaction gap for children with autism. By combining **Generative AI (DreamGaussian)** and **Augmented Reality (AR)**, our system transforms children's hand-drawn 2D sketches into personalized, interactive 3D companions.

Unlike traditional therapy tools with fixed avatars, AutismRecover empowers children to interact with their own creations. Through intuitive gestures like high-fives and emotionally supportive dialogue, the system creates a safe, immersive environment that fosters communication and confidence.

## ğŸ—ï¸ System Architecture

The system is built upon four core layers:

1. **User Interaction Layer**: Handles drawing input and the visualization of the cartoon world.
2. **Conversational Layer**: Manages speech bubbles and companionship logic.
3. **Interactive Connection Layer**: Connects the virtual canvas with 3D character interactions.
4. **Core System Layer**:
   * **Generative AI**: Sketch processing, 3D rendering (DreamGaussian), and behavior generation.
   * **Data & Feedback**: Content scheduling and loading prompt management.
